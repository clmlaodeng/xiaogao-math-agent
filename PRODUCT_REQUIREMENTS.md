# 成都小升初 AI 教研助手 MVP 产品需求

版本：MVP v0.1

## 1. 产品目标

用最小可用版本验证一件事：

> 成都数学老师或小型教培机构是否愿意为“自动出题、自动组卷、错题变式、学情分析、家长反馈”持续付费。

第一版不追求全自动替代老师，而是做“老师审核型 AI 教研工作台”。

## 2. 目标用户

### 主要用户

成都小升初数学老师、小型教培机构教研负责人。

### 次要用户

六年级家长，但第一版不单独做家长端登录。

## 3. MVP 范围

### P0 必须做

1. 按薄弱点自动出题
2. 错题变式训练
3. 自动组卷
4. 家长反馈生成
5. 导出 Word/PDF

### P1 暂缓

1. 学生登录
2. 在线答题
3. 自动拍照批改全流程
4. 班级长期学情看板
5. 机构多账号权限

### P2 不做

1. 自训练大模型
2. 自研 OCR
3. 排课系统
4. 支付系统
5. 大而全题库平台

## 4. 页面结构

### 页面 1：首页工作台

展示：

- 今日待生成材料
- 最近生成的练习
- 高频薄弱点
- 快捷入口

快捷入口：

- 生成专项题
- 错题变式
- 生成周测卷
- 生成家长反馈

### 页面 2：生成专项题

输入字段：

- 年级：默认六年级
- 知识点
- 薄弱点
- 题量
- 难度：基础 / 中档 / 拔高
- 是否成都风格

输出字段：

- 题目
- 答案
- 解析
- 考点
- 易错点
- 训练价值

验收标准：

- 生成 5 道题不超过 30 秒。
- 每道题必须包含答案和解析。
- 解析不能使用初中超纲方法。

### 页面 3：错题变式

输入字段：

- 原题文本
- 学生错误答案，可选
- 老师补充说明，可选

输出字段：

- 原题考点
- 可能错因
- 3 道同模型变式
- 1 道基础补救题
- 1 段家长反馈

验收标准：

- 变式题不能只是改数字，必须保持同一数学结构。
- 家长反馈不能超过 180 字。
- 反馈要包含“问题 + 原因 + 下一步练习建议”。

### 页面 4：自动组卷

输入字段：

- 卷子类型：专项小测 / 周测 / 课后巩固
- 题量
- 时间
- 知识点列表
- 难度比例

输出字段：

- 学生卷
- 答案卷
- 讲评提纲
- 易错点汇总

验收标准：

- 支持导出 Word/PDF。
- 卷面结构清晰，老师能直接发给学生。
- 答案卷和学生卷分开。

### 页面 5：家长反馈

输入字段：

- 学生姓名
- 本周薄弱点
- 本周进步点
- 错因
- 下周训练安排

输出字段：

- 微信简短版
- 正式报告版

验收标准：

- 微信版不超过 200 字。
- 正式报告版不超过 600 字。
- 语气专业、克制，不制造焦虑。

## 6. AI 输出规则

所有题目输出必须包含：

- 题目
- 答案
- 解析
- 考点
- 易错点
- 适合学生

所有解析必须遵守：

- 默认小学五六年级方法
- 不使用方程组等明显超纲方法，除非用户明确要求
- 关键步骤不能跳
- 如果题目质量不确定，要提示老师审核

家长反馈必须遵守：

- 不夸大问题
- 不承诺提分
- 不使用“择校押题”等违规表述
- 以训练建议和学习习惯为主

## 7. 第一版成功标准

产品层面：

- 老师能在 10 分钟内生成一份可用周测卷。
- 老师能在 3 分钟内把一道错题变成一组训练题。
- 家长反馈可以直接复制到微信。

商业层面：

- 5 个真实老师试用。
- 2 个老师连续使用 2 周。
- 1 家机构愿意付费试点。

质量层面：

- 生成题目 80% 以上无需大改。
- 解析错误率低于 5%。
- 家长反馈无需大改比例达到 80%。
