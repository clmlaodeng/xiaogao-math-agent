# 用户入口与产品话术

这个 agent 想有市场价值，入口必须比“请描述你的需求”更具体。用户越不用思考，越容易留下来用。

## 1. 首页一句话

> 成都小升初数学专项出题助手：输入孩子薄弱点，自动生成成都风格训练题、详细解析和同模型变式。

## 2. 开场白

我是您的成都小升初数学训练助手。您可以直接说孩子哪里不会，我会自动生成对应的成都风格专项题，并配详细解析、易错提醒和后续变式训练。

您也可以这样问：

- 出 3 道中档量率对应应用题，成都风格，带解析。
- 孩子总是找错单位 1，帮我专项练一下。
- 给我一组差不变原理题，从基础到拔高。
- 仿成都私立升班考风格，出一套 30 分钟小测。
- 我发一道错题，你帮我生成 3 道同模型变式。

## 3. 快捷按钮建议

适合做成产品里的按钮：

- 按薄弱点出题
- 按考点出题
- 上传错题生成变式
- 生成 30 分钟小测
- 生成一周训练计划
- 只要题干
- 需要详细解析

## 4. 典型用户路径

### 家长路径

用户输入：

> 孩子分数应用题老是不会找单位 1。

agent 输出：

- 判断薄弱点：单位“1”识别
- 生成 3 道中档题
- 每题详细解析
- 每题易错提醒
- 推荐下一步：基础巩固或限时训练

价值：

家长不用懂教研，也能拿到针对性练习。

### 老师路径

用户输入：

> 我要上差不变原理复习课，帮我做一组例题和变式。

agent 输出：

- 教学目标
- 例题 2 道
- 变式 4 道
- 易错题 2 道
- 答案解析
- 课堂追问

价值：

老师节省备课和改题时间。

### 机构路径

用户输入：

> 生成一套成都小升初应用题周测，量率对应为主，30 分钟。

agent 输出：

- 试卷结构
- 分值
- 题目
- 答案解析
- 能力覆盖表

价值：

机构可以稳定生产教研材料。

## 5. 付费价值设计

免费版可以提供：

- 每次生成 3 道题
- 基础解析
- 常见考点训练

付费版可以提供：

- 上传错题生成同模型变式
- 生成整套试卷
- 生成答案卷和讲解稿
- 按学生水平生成一周训练计划
- 按成都私立名校风格生成专题卷
- 老师批量生成题组
- 学习报告和错因分析

## 6. 对外宣传文案

短版：

> 不用翻题库，不用筛难度。告诉我孩子薄弱点，自动生成成都小升初风格专项训练题。

中版：

> 这不是普通搜题工具，而是一位成都小升初数学教研助手。它能根据孩子薄弱点生成专项题，按成都私立升班考风格控制难度，并提供小学方法详细解析和同模型变式训练。

老师版：

> 让教研组长帮你批量改题。输入考点和难度，快速生成同模型、不同数据、带解析和易错点的训练题。
